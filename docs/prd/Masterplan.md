# 🏁 Masterplan: Must Be The Apex

## 1. 프로젝트의 존재 이유 (Why)
현재 F1 25 및 심레이싱 유저들은 주행 데이터를 분석하기 위해 월 구독료가 발생하는 고가의 텔레메트리 서비스를 이용하거나, 분석하기 어려운 원시 데이터를 직접 해석해야 하는 장벽에 부딪히고 있습니다. 
**Must Be The Apex**는 이러한 진입 장벽을 허물고, 누구나 **무료로 로컬 환경에서 평생 사용 가능한 n8n 스타일의 셀프 호스팅 텔레메트리 플랫폼**을 제공하기 위해 존재합니다. 나아가, 단순 수치 제공을 넘어 **AI 레이스 엔지니어**가 유저의 주행 데이터를 분석하여 실질적이고 물리적인 조작 가이드를 제공함으로써 드라이버의 실력 향상을 돕습니다.

## 2. 최종 제품의 형태 (What)
- **Windows Agent (초경량 데이터 전송기):** 웹 브라우저는 UDP 패킷을 직접 수신할 수 없으므로, 게임이 실행되는 PC에서 F1 25의 UDP 패킷을 수신하고 정제하여 대시보드 서버로 쏴주는(Forwarding) 역할을 합니다. 도커가 필요 없는 단일 실행 파일(`.exe`) 형태로 제공되어 매우 가볍게 동작합니다. (f1laps 에이전트와 유사한 범용 전송기 역할)
- **Web Dashboard (Docker 기반 셀프 호스팅 뷰어 & AI):** React Router + Remix 기반의 웹 서버와 SQLite DB를 포함합니다. 이 대시보드 팩은 완벽하게 **Docker**로 패키징되어, 유저가 게임 PC 자체에서 돌리거나 개인 NAS, 또는 클라우드 어디에든 손쉽게 띄울 수 있는 진정한 셀프 호스팅을 지원합니다.
- **Mobile Handoff (확장성):** PC 대시보드에서 제공하는 QR 코드를 스캔하여, 스마트폰/태블릿을 스티어링 휠 전용 미니 대시보드로 즉시 활용할 수 있는 모바일 뷰.
